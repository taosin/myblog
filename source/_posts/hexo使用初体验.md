# hexo初体验
## hexo终于还是搭好了，分享一些踩坑经验
### 安装Hexo
#### 首先在本地创建目标目录，
#### 使用下面命令安装hexo
```
mkdir hexo & cd hexo
```
```
sudo npm install -g hexo
```
### hexo初始化
```
hexo init
```
生成静态页面

```
hexo generate（hexo g 等同于 hexo generate）
```
### 启动hexo
```
hexo server (hexo s 等同于 hexo server)
```
浏览器地址栏输入http://localhost:4000 ,即可浏览页面，此时hexo已经安装成功。
